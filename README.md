# PML88 Football Companion Bot

An interactive, API-free Telegram bot built with Node.js and Telegraf. Designed for football enthusiasts, PML88 provides offline tactical breakdowns, legendary fan chants, and random football trivia challenges.

## Features
- **Tactical Playbooks:** Deep dives into historic football strategies (Gegenpressing, Tiki-Taka, Low Block).
- **Stadium Fan Chants:** Randomly generates classic club and match-day crowd anthems.
- **Football Trivia:** Instant quiz module testing historical football facts.

## Deployment on Railway
1. Push this repository to your GitHub account.
2. Log into **Railway.app** and connect this repository to a new project.
3. In the Railway project **Variables** tab, add your Telegram Bot Token as `BOT_TOKEN`.
4. Deploy! Railway will automatically detect the configuration and bring `t.me/PML88BOT` to life.
